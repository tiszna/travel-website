# Travel Website

TravelLogs is a travel inspiration and planning website that showcases curated itineraries for different regions. Users can explore destinations, view trip details, and plan their adventures with ease.

## Features
- Homepage with featured destinations and highlights
- Regional pages (Asia, Europe, Americas, Oceania)
- Detailed itinerary pages for specific trips (e.g., Bali)
- Contact form for inquiries
- Responsive design with HTML & CSS

## Viewing the Website
- **Live:** [TravelLogs on GitHub Pages](https://tiszna.github.io/travel-website/)
- **Locally:** Open `index.html` in any web browser.

## Technologies Used
- HTML5
- CSS3
- Git & GitHub
